# 表现与音效 · FX

## 职责
提供视觉特效与音效，强化"镜子"主题，覆盖倒影、激光、反射、命中、镜像同步与 UI/BGM。

## 分类 / 做法
- **地面倒影**：Planar Reflection（**纯视觉特效**，不影响玩法），强化镜面主题。
- **激光束 VFX**：激光塔射线的视觉表现（含反射后的光路段）。
- **反射光路**：反射镜改向后的光路可视化。
- **命中特效**：伤害命中单位/据点的粒子反馈。
- **镜像同步动画**：复制镜投影与原件的动画/攻击同步表现（视觉层面）。
- **音效**：UI 音效、开火/命中/建造/镜子放置等。
- **BGM**：背景音乐。

## 关键参数
> 全部为 Godot `@export`，编辑器运行时可调。

| 参数名 | 默认值 | 说明 |
|---|---|---|
| reflection_opacity | 0.4 | 地面倒影不透明度 |
| laser_width | 0.1 | 激光束宽度 |
| hit_fx_scale | 1.0 | 命中特效缩放 |
| sfx_volume | 0.8 | 音效音量 |
| bgm_volume | 0.6 | 背景音乐音量 |

## 关键架构
```
PlanarReflection: 地面反射相机 + RenderTexture（纯视觉）
LaserVFX: 沿 Mirror 求解出的光路段绘制 beam
HitFX / MirrorSyncFX: 事件触发的粒子/动画
AudioManager: SFX 池 + BGM 播放
（表现层订阅逻辑事件，不反向影响玩法）
```

## 已知限制 / 初版不做的部分
- 倒影为 Planar Reflection 纯视觉，不做真实镜面 SSR/光追。
- 不做动态天气/昼夜/后处理调色管线。
- 音效初版为占位资源，不做动态混音/空间音频。
