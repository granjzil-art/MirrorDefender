# MirrorDefender · 设计基线（已确认）

> 本文件是所有决策的**唯一事实源**。用户已于 2026-07-16 拍板确认（含二次细化修正），后续实现严格以此为准。
> 变更需走 DECISIONS 流程记录。版本：**v1.1**（游戏定名 **MirrorDefender**）

---

## 0. 游戏定性（已确认）

| 项 | 决策 |
|---|---|
| 类型 | **经典塔防**（波次分段），**取消双据点对攻/阵营对峙** |
| 胜负 | **坚持固定波数即胜**；我方据点 HP 归零即败 |
| 视角 | 斜俯视 3D |
| 核心机制（初版） | **复制镜 + 反射镜（镜面反射激光）**，仅此两个 |
| 技术栈 | Godot 4.x + GDScript，KayKit Medieval Hex 资产 |
| 网格 | **初版同时支持六边形 + 正方形**，各做一个示意关卡 |

## 1. 镜像与机制定性（已确认）

- 复制镜沿生效侧法线逐格搜索，只复制**最近的一个非空地块**中全部可复制内容；普通空地不终止搜索。
- 可复制内容为地块建筑、地块屏障、大石头/尖刺/空洞等地块效果和已有虚像；敌人、镜子及其它边建筑不复制。
- 镜像默认**不占格**，可落在不可建造地形、与真实内容或多个虚像叠加；`projection_ignores_occupancy` 关闭时才启用严格投影占位。
- 镜像建筑无独立 AI、无独立索敌、无独立动画时钟；原件索敌/开火时镜像才同步攻击。朝向和攻击点经镜像变换，**允许打空**。
- 屏障虚像共享原屏障耐久，虚像承伤转交原件；地块效果虚像在投影格执行相同导航/伤害/死亡效果。
- 镜像**可被再次复制**；来源链禁止重复经过同一镜子，`copy_chain_max` 默认 4。
- 单位**不可被镜像**（初版）；**无中立单位**。
- 复制源使用统一 payload 契约；后续新增格建筑、陷阱、障碍或机关无需在 MirrorManager 按类型硬编码。

## 2. 高低差（已确认）

- 离散**档数可配置**（默认 3 档，编辑器可改）。
- 只立**基本物理规则**：**底层激光会被隆起地形遮挡**。
- 高度服务镜面主题（承接原专题结论）。

## 3. 天空层 / 倒影（已确认）

- **天空层玩法：初版不做**，列为后续扩展。
- **地面倒影：作为纯视觉特效保留**（雨后水坑观感）。

## 4. 镜子放置规则（已确认 · 关键）

- 镜子**沿六边形/正方形的边放置，严格贴合**；**不受地块归属限制，任意边可放**。
- 镜子是**单面**的（只有朝向正确的一面生效）。
  - 六边形：**6 种朝向**；正方形：4 种朝向。
- 复制镜：生效面决定沿哪一侧的法线搜索最近非空地块；翻面后立即重算另一侧的来源和投影。
- 不同镜子朝向决定**镜像对称轴**（详见机制文档）。
- **镜长 = 边长**（整条边变成镜面）；**镜子占用这条边，一条边至多一面镜**（相邻两格共享该边，任一格占用后另一侧不能再放）。
- **鼠标交互**：悬停/选中格子或边时**高亮显示**，辅助精确贴边放置。

## 5. 各系统初版范围（已确认）

| 系统 | 初版决策 |
|---|---|
| Grid | 六边形 + 正方形两种，各一示意关卡 |
| 地块 | **传统塔防**，无阵营。分类：①可建造 ②可破坏后建造(石头等障碍) ③不可破坏不可建造(路面)。高度离散3档。做**拖拽式地块编辑器**（插槽填充预制地块 + 单地块参数修改） |
| 建筑 | 初版：**箭塔** + **激光塔(无限穿透单位, 对触碰敌人持续伤害)**。朝向可调且**不随镜头角度变化**：六边形关卡 **6 向**(始终垂直于边)，四边形关卡 **8 向**(垂直四边 + 4 对角线) |
| 单位 | 不可被镜像，无中立单位 |
| 资源 | **单一主资源**；建筑与镜子共用、各有上限；产出来源全做但**带开关**（击杀掉落/占领产出/生产建筑/时间增长/破坏地块） |
| 波次 | 经典塔防波次；每波构成需详细参数（怪种/出怪频率/出生点/路径等） |
| 索敌 | 无伤害类型克制；伤害 = **固定值(塔种决定) × 等级增伤因子 × 额外增伤因子**；索敌参数全做 |
| 寻路 | **不做自动寻路**，路径由设计者手动指定；支持六边形/正方形 |
| 镜子 | 初版：复制 + 激光反射 |
| 相机/输入 | WASD 移动镜头、QE 旋转镜头、XC/滚轮缩放、**R 旋转塔/镜子朝向**、鼠标操作 |
| UI/HUD | 沿用原型；顶部血条改为 **我方据点血量 \| 本波剩余敌人 x/y** |
| AI | 纯脚本波次：出生点生成 → 按行为模式沿指定路径行走 → 触据点则消失并造成伤害 |
| 存档/关卡 | 数据资源驱动，与逻辑解耦 |
| 表现/音效 | 按原方案。**美术初版用色块 / 点线面替代**（镜=线段、塔/敌=几何体、激光=发光线）；HexTileset 仅备用资产，未必采用 |

## 6. 配置理念（已确认 · 硬规范）

> **如无必要，一切配置直接在 Godot 编辑器内完成**（`@export` 运行时可调），不做外部配置表。仅当数据量大到编辑器不便时才用数据资源(.tres)。

## 7. 二次拍板细化（2026-07-16 · 全部已确认）

| 主题 | 结论 |
|---|---|
| 镜像占格与落点 | 虚像默认不占格，可落在非建造地形并叠加；严格占位由 `projection_ignores_occupancy` 反向开关 |
| 镜像攻击朝向 | 由原件朝向经镜像变换决定，**不索敌、允许打空** |
| 复制源 | 沿生效侧法线最近的一个非空地块，复制该格全部地块建筑/元素/虚像 |
| 镜像可否再复制 | **可**；来源链禁止重复镜子，`copy_chain_max` 默认 4 |
| 反射镜背面 | 做成开关 `back_face_mode`：**PASS 穿过 / BLOCK 阻挡**（阻挡表现同地形），编辑器实测选择 |
| 建筑是否挡光 | **不挡**，激光穿过建筑 |
| 镜子空间占用 | 一条边至多一面镜；镜长 = 边长 |
| 建筑朝向档位 | 六边形 6 向(垂直边)、四边形 8 向(垂直边 + 对角线)；**不随镜头旋转** |
| 垂直入射反射 | 反射光沿切向微偏移 `perp_offset`，保证与入射光平行但不重叠 |
| 端点命中 | 镜=整条边线段，命中按线段区间 `t∈[0,1]` 常规判定，无特殊分支 |
| 激光伤害 | 对触碰敌人**持续伤害**（每 tick 结算），非瞬伤 |
| 鼠标交互 | 悬停/选中格子或边高亮显示 |
| 美术 | 初版色块/点线面替代，HexTileset 备用 |
| 六边形/正方形对称轴 | 六边形 flat-top；正方形对称轴 = 边所在直线（沿用） |

> 至此初版设计无阻塞项，可进入实现（M1）。
