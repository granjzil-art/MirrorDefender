# Mirror 塔防 · 扩展机制设计

> 在你给出的三大机制（复制塔 / 折射双防 / 基底映射）基础上，围绕"镜子 = 对称 / 复制 / 反射 / 映射"这一动词内核继续衍生。
> 版本：v0.1 · 2026-07-15
> 标注：★ 推荐优先做 · ◆ 进阶 · ○ 长线/DLC

---

## A. 基于"边缘类"的衍生（镜子作为边）

- **★ A1 镜像回廊（多边串联）**：相邻的多面镜子边可串成一条"镜廊"，单位沿镜廊被逐格复制，形成一整排复制军团。用于爆发推进，代价是镜子消耗大。
- **◆ A2 单向镜（半透镜）**：一面只从一侧生效的镜子。你的复制单向输出到敌方向，敌人的镜像不会反打回来。引入"朝向"这一策略维度（呼应你 3C 里的 QE 旋转）。
- **◆ A3 镜子摧毁与反制**：敌方可派"碎镜者"单位破坏你的镜子边，镜子被破瞬间所有镜像体消失——制造攻防博弈点。
- **○ A4 棱镜分光**：折射机制进阶，一束激光过棱镜镜子后分成多束不同颜色（对应不同伤害类型：物理/穿透/灼烧），做元素克制。
- **◆ A5 汇聚镜（凹面镜）**：多束激光被凹面镜汇聚到一点，形成超高单点伤害的"聚焦烧点"，克制高血量 Boss。

## B. 基于"基底类"的衍生（镜子作为面/层）

- **★ B1 双层战场（天/地）**：把你的天空镜像层做成可放塔的**独立战场层**。地面走地面兵，天空走镜像兵，玩家要同时经营两层，塔可选择打本层或对穿另一层。这是把"映射"升级成核心策略层的关键。
- **◆ B2 层间传送门**：镜面地块作为上下层的"电梯口"，单位可主动切层，绕过本层的封锁。攻方用它突袭，守方要预判封堵。
- **◆ B3 镜像世界的资源**：倒影层里藏有"虚像资源"，只能用镜像单位采集，逼玩家经营镜像军队而不只是复制打架。
- **○ B4 昼夜/明暗镜**：镜面在明/暗环境下映射不同的层（白天映天空、夜晚映地下），做关卡节奏变化。

## C. 全新动词：把"对称"玩出花

- **★ C1 对称建造（镜像蓝图）**：玩家在一侧建造，系统自动沿镜面对称轴在另一侧生成对称的一份。快速铺开防线的爽点，也是"镜面"最直觉的操作反馈。
- **◆ C2 镜像反伤（属性反射）**：一种防御镜塔，把受到的伤害按比例反射回攻击者。呼应"镜子把东西弹回去"的直觉。
- **◆ C3 镜像陷阱（假目标）**：生成一个吸引敌方索敌的镜像诱饵（无实体伤害的投影），操纵敌人走位——把"投影不是实体"这个设定变成正向玩法而非困惑点。
- **○ C4 破碎镜像（限定技）**：主动击碎一面大镜，把当前所有镜像体"实体化"为一次性真实军队，打一波总攻。高风险高回报的终结技。

## D. 与"高低差"结合（承接 01 文档专题）

- **◆ D1 高度镜像塔**：地面塔的天空镜像变成俯射塔，覆盖范围与地面互补，形成立体火力网。
- **◆ D2 镜像桥**：天空镜像层是平的，用来跨越地面河流/断崖，"高低差制造问题、镜子提供解法"。
- **○ D3 落差聚光**：激光沿高度台阶折射，做立体解谜式关卡。

## 推荐落地顺序（MVP → 完整）

| 阶段 | 纳入机制 | 目标 |
|---|---|---|
| MVP（验证核心） | 复制塔(用户1) + 对称建造(C1) | 验证"镜子=复制战力"是否好玩 |
| V1（成型） | + 折射双防(用户2) + 镜像陷阱(C3) + 单向镜(A2) | 攻防博弈成型 |
| V2（策略深度） | + 双层战场(B1) + 层间传送门(B2) + 高度镜像(D1) | 立体策略 |
| 长线 | 棱镜分光/汇聚镜/破碎镜像等 | 内容扩展 |

> 所有机制都遵循同一个内核动词，玩家学习成本低；每个都可作为独立模块开关，符合你的"模块化 + 可拓展"要求。
