# MirrorDefender · 初版执行计划与验收标准

> 协作四步流程第 3 步。范围严格限定初版：复制镜 + 反射镜 + 六边形/正方形网格 + 两个示意关卡。
> **2026-07-16 二次拍板：全部设计细节已确认，无阻塞项，可从 M1 开工。** 版本：v1.1 · 游戏定名 **MirrorDefender**

---

## 一、初版范围（Scope）

**做**：网格(六边形+正方形) · 地块(3类+编辑器+3档高度) · 相机输入(WASD/QE/XC/R) · 建筑(箭塔+激光塔) · 敌方单位 · 手动路径 · 波次系统 · 索敌战斗(固定伤害公式) · 资源(单一+上限+产出开关) · **复制镜 + 反射镜** · UI(原型布局+血条改造) · 脚本AI · 关卡数据 · 地面倒影特效 · 两个示意关卡。

**不做**：天空层映射、单位镜像、伤害类型克制、自由格网寻路、双据点对攻。大石头/尖刺/空洞等地块内容已纳入 M5 复制范围。

## 二、里程碑与任务拆解

### M1 · 网格与相机地基
- Grid 抽象层：六边形(flat-top立方体坐标) + 正方形，边(Edge)可挂载点。
- 相机：WASD 平移、QE 旋转、XC 调俯仰、仅滚轮缩放、斜俯视。
- **验收**：能生成两种网格地图；相机可自由观察；点击可拾取到具体格子与具体边。

### M2 · 地块系统与地块编辑器
- 3 类地块 + 离散高度(档数可配)。
- 拖拽式地块编辑器：调色板拖拽预制地块填充、单格改参。
- **验收**：能在编辑器中拖拽搭出一张地图并保存为关卡资源；可破坏障碍格→变可建造。

### M3 · 建筑 + 索敌战斗 + 资源
- 箭塔(单体索敌)、激光塔(固定方向射线, 穿透单位, **对触碰敌人持续伤害**, 不被建筑阻挡)、路径屏障(三级耐久、脱战回血、可配置反伤)。
- 伤害公式：固定值×等级因子×额外因子；索敌策略可选。
- 资源：单一主资源、建筑/镜子上限、5 种产出带开关。
- R 键旋转建筑朝向：**六边形 6 向 / 四边形 8 向，档位固定于世界坐标不随镜头变化**。
- **验收**：放箭塔能索敌开火按公式扣血；激光塔射线穿透多个敌人并持续掉血；建筑不挡光；朝向按 6/8 档旋转且不随镜头改变；资源正确增减、达上限阻止建造。

### M4 · 单位 + 手动路径 + 波次 + AI
- 敌方单位沿手动指定路径移动，触据点消失并扣据点血。
- 波次：详细出怪参数(怪种/数量/频率/出生点/路径)。
- **验收**：能配置一波怪按参数刷出、沿路径走到据点扣血；坚持到设定波数判胜、据点血空判负。**已完成：关卡编辑器的路径/波次页与 M4DemoLevel 两波验收资源已交付；仅第一波手动开始，全部出怪组按相对首次点击的全局延迟自动执行；近战与弓箭手会停止攻击路径屏障，屏障摧毁后继续前进。**

### M5 · 复制镜（核心）
- 沿边严格贴合放置；单面朝向；R 键翻面切换生效侧；**一条边至多一面镜，镜长=边长**。
- 从生效侧紧邻格沿边法线逐格搜索，只选取最近的一个非空地块，复制其全部地块建筑、地块元素和虚像。
- 虚像默认不占格并允许多个叠加；可投影到大石头、尖刺、空洞等不可建造格。`projection_ignores_occupancy` 可切换严格投影占位。
- 箭塔/激光塔虚像无独立 AI/索敌/动画时钟，原件攻击时才同步攻击；屏障虚像共享原件耐久；地块元素虚像执行相同效果。
- 镜像朝向由原件朝向经镜像变换决定，**不索敌、允许打空**。
- 对称轴=镜子所在边；镜像位置/朝向做关于轴的反射。
- **镜像可被再次复制**；镜子不可复制；来源链禁止重复经过同一镜子，`copy_chain_max` 默认 4。
- 边放置预览同时显示镜面、生效侧、源格、落点格、复制体类型/位置和失败原因。
- 悬停/选中格子或边高亮。
- **验收**：见下 §三。

### M6 · 反射镜（核心）
- 沿边放置；单面；激光反射(反射向量公式)、多次反射(上限)、被障碍/隆起地形/据点阻挡。
- **背面模式开关** `back_face_mode`：PASS 穿过 / BLOCK 阻挡(表现同地形)，编辑器可切。
- **建筑不挡光**；激光对触碰敌人持续伤害。
- **垂直入射偏移** `perp_offset`：入射光与反射光平行不重叠。
- **验收**：见下 §三。

### M7 · UI + 表现 + 两个示意关卡
- HUD：资源栏、建造栏、检视/升级面板、小地图、操作提示；顶部改为【我方据点血量 | 本波剩余敌人 x/y】。
- 地面倒影特效、激光VFX、镜面对称轴调试线。
- 交付六边形关卡 ×1、正方形关卡 ×1。
- **验收**：两关可完整游玩通关；UI 信息准确；镜子机制在关卡中可用且直观。

## 三、核心机制验收标准（重点）

### 复制镜
1. 只能贴边放置，六边形6条边/正方形4条边均可放；共享边两侧地块都需允许边建筑。 ✓
2. **一条边至多一面镜**；镜长=边长（相邻两格共享边，占用后另一侧不能再放）。 ✓
3. 单面：从生效侧紧邻格沿法线搜索最近非空地块；R 翻面后重算另一侧。 ✓
4. 一面镜子只选一个源格，但复制该格全部地块建筑、地块元素和已有虚像；镜子/其它边建筑不复制。 ✓
5. `projection_ignores_occupancy=true` 时虚像不占格，可落在石头/尖刺/空洞且同格叠加；关闭后真实建筑或其它投影组会阻止生成。 ✓
6. 建筑虚像不可单独选中，无独立 AI/索敌/动画时钟；攻击朝向/落点由原件变换得到，允许打空。 ✓
7. 原件索敌开火时，全部派生塔虚像同帧开火；地块元素虚像在投影格执行相同效果。 ✓
8. 屏障虚像可阻挡敌人，承伤转交原屏障并共享耐久；原件销毁时派生链立即消失。 ✓
9. 大石头虚像进入现有动态换路判定；尖刺/空洞虚像保留原效果与 `affects_airborne`。 ✓
10. 换不同边或改变源格距离后，镜像落点/朝向按轴对称正确重算。 ✓
11. 镜像可再被另一面镜子复制；来源链不重复镜子，`copy_chain_max` 截断链长。 ✓
12. 边放置预览可见镜面、生效侧、源格、目标格、复制体类型/位置和失败原因；R 翻面实时刷新。 ✓
13. 虚像保留原物轮廓，使用统一半透明、自发光和层级偏移表现。 ✓

### 反射镜
1. 只能贴边放置，单面生效；一条边至多一面镜。 ✓
2. 激光命中生效面按反射定律改向。 ✓
3. **背面模式开关**：PASS 穿过 / BLOCK 阻挡(表现同地形)，编辑器可实时切换。 ✓
4. 支持多次反射，达 `reflect_max` 截断（防闭环）。 ✓
5. 激光被障碍物、隆起地形(高于激光层)、据点、边界阻挡而停止；**建筑不阻挡**。 ✓
6. 激光对路径上所有敌方单位造成**持续伤害**（每 tick 结算，穿透单位）。 ✓
7. **垂直入射时**反射光沿切向 `perp_offset` 微偏移，与入射光平行不重叠。 ✓
8. 端点命中按线段区间 `t∈[0,1]` 常规判定，无特殊分支。 ✓
9. 反射光路与入射重叠段产生叠加伤害（涌现，无需特判）。 ✓
10. 所有关键参数(reflect_max、back_face_mode、perp_offset、激光层高度等)可编辑器运行时调。 ✓
