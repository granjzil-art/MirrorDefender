## 一、游戏概述

| 项 | 内容 |
|---|---|
| 游戏名 | **MirrorDefender** |
| 类型 | 经典塔防（波次分段），主题 **Mirror（镜子）** |
| 视角 | 斜俯视 3D |
| 胜负 | **坚持到固定波数即胜**；我方据点 HP 归零即败（无敌方据点、无阵营对攻） |
| 核心卖点 | 把"镜子"做成**机制级玩法**：镜子作为几何图形的**边**，既能**复制**建筑火力、又能**反射**激光光路 |
| 技术栈 | Godot 4.x + GDScript |
| 美术 | **初版用色块 / 点线面替代**（镜=线段、塔/敌=几何体、激光=发光线）；HexTileset 为备用资产 |
| 网格 | 初版同时支持**六边形(flat-top)** 与 **正方形**，各做一个示意关卡 |

### 核心循环
准备期布防（建塔/放镜子/调朝向）→ 波次来袭（敌人沿指定路径推进）→ 塔与镜子协同拦截 → 波次结算/资源产出 → 下一波 → 坚持到目标波数判胜。

## 二、两大核心机制（初版仅此两个）

### 1. 复制镜 CopyMirror
- 镜子贴在地块的**边**上，生效侧朝向有建筑的格 → 把该建筑沿"边所在直线"**镜像**出一个**投影**。
- 投影**占格**、无独立 AI、不索敌；**原件开火时投影同步开火**，朝向由原件朝向做镜像变换 → **可能打空**（忠实倒影）。
- **落点校验**：图外 / 被建筑/镜像/障碍/地形占据 → **不生成**。
- **镜像可被再次复制**（规则一致，先做，性能问题用 `copy_chain_max` 截断）。
- 预留 `ICopyable`：后续可复制石头等非建筑物。

### 2. 反射镜 ReflectMirror
- 镜子贴边，激光命中**生效面**按反射定律改向，可**多次反射**（`reflect_max` 封顶）。
- **背面模式开关** `back_face_mode`：PASS 穿过 / BLOCK 阻挡（阻挡表现同地形）——编辑器实测选择。
- 激光被**障碍/隆起地形/据点/边界**（及 BLOCK 模式镜背面）挡停；**建筑不挡光**。
- 激光对光路上**所有触碰敌人施加持续伤害**（每 tick 结算，穿透单位）。
- **垂直入射偏移** `perp_offset`：保证入射光与反射光平行不重叠。

### 镜子通用空间规则
- **镜长 = 边长**（整条边即镜面）；**一条边至多一面镜**（相邻两格共享边，占用后另一侧不可再放）。
- **单面朝向**：六边形 6 种朝向 / 正方形 4 种；`R` 键翻面切换生效侧。
- 对称轴 = 镜子所在边所在直线。
- 悬停/选中格子或边**高亮显示**。

> 详见 `systems/Mirror_镜子系统.md`（含几何公式、边界处理、参数表）。
