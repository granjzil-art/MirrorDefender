# Mirror 塔防 · 界面原型说明

> 交互原型：`Docs/prototype/index.html`（浏览器打开，支持 QE 旋转 / XC 缩放键盘交互）
> 版本：v0.1 · 2026-07-15

## 设计目标与对应
| 你的要求 | 原型如何满足 |
|---|---|
| 清晰展示局势战况 | 顶部**双据点血条 VS 视图**一眼看清攻防差；地块用己方绿/敌方红/中立/河流蓝分色 |
| 扩张成就感 | 地块按归属着色，占领即变色；小地图俯瞰整体版图推进；资源栏实时增长 |
| 地图展示角度 | **斜俯视 3D（rotateX 52°）**，兼顾"看清战场"与"立体感"，带高度台阶侧壁表现高低差 |
| UI 可读易用 | 底部**卡片式建造栏**（图标+名称+造价+机制色点）；右侧**检视/升级面板**（数值+等级点+升级/出售）；左侧机制图例 |
| 镜子机制呈现 | 三种机制在战场中直接可视化（见下）+ 地面倒影层强化镜面氛围 |

## 三种镜子机制的可视化
1. **边镜·复制塔**（红色粗边）：本体蓝单位 → 经公共边镜面 → 相邻格出现半透复制体。
2. **边镜·折射**（橙色光线）：激光射向红色镜面边 → 反射折向 → 形成第二段防线。
3. **面镜·映射**（蓝色渐变格 🌀）：河流格设为面镜 → 单位上浮到**天空层镜像**（半透倒悬）→ 跨越河流。

## 视觉主题落地
- **地面倒影层**：战场下方镜像一份、上下翻转、模糊渐隐——像"雨后水坑"，强化 Mirror 主题但不干扰玩法（纯表现，见 01 文档结论）。
- 配色为深蓝夜幕 + 高饱和地块，据点用金/红描边聚焦视线。

## 交互
- `Q`/`E` 旋转战场，`X`/`C` 缩放（演示 3C 中相机操作手感）。
- 建造栏卡片、升级/出售按钮均为可点击态样式。

> 这是**演示局势与 UI 布局的原型**，非最终美术。KayKit 3D 模型将在实现阶段替换占位表现。
